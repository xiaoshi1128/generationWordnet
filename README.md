# generationWordnet
Generalization tree by using wordnet   
First you should install WordNet and some jar files that you need.


Like this:
<GeneralizationTree>
  <entity>
    <physical_entity>
      <matter>
        <solid>
          <solid_food>
            <garden_truck>
              <citrous_fruit>
                <orange#n#1/>
              </citrous_fruit>
            </garden_truck>
          </solid_food>
        </solid>
      </matter>
    </physical_entity>
    <abstract_entity>
      <abstraction>
        <attribute>
          <property>
            <visual_property>
              <spectral_colour>
                <orange#n#2/>
              </spectral_colour>
            </visual_property>
          </property>
        </attribute>
      </abstraction>
    </abstract_entity>
